# CodeAlpha_ChatbotForFAQs
An intelligent FAQ Assistant built with Python, scikit-learn, and Tkinter. It uses TF-IDF vectorization and Cosine Similarity to match user questions with accurate responses in an interactive chat interface.

# 🤖 FAQ Assistant Chatbot
An intelligent FAQ Assistant built using **Natural Language Processing (NLP)** and **Tkinter** to help users instantly answer queries regarding internship rules, project submissions, and certificate requirements.

---

## 🚀 Features
* **TF-IDF Vectorization:** Converts questions into mathematical term-frequency vectors using `scikit-learn`.
* **Cosine Similarity Engine:** Matches user queries against stored FAQs based on semantic similarity.
* **Confidence Threshold:** Includes fallback handling for unrecognized queries to prevent incorrect answers.
* **Interactive UI:** Built using `tkinter` with customized chat bubble styling and real-time response scrolling.

---

## 🛠️ Tech Stack
* **Language:** Python 3.13
* **NLP Library:** `scikit-learn`
* **GUI Framework:** Tkinter
